Html

Hypertext markup language(HTML) is the standard markup language for creating web pages and web applications. Web browsers receive Html documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originaly includes cues for the apperance of the document.

HTML elements are the building blocks of HTML elements are the building blocks of HTML pages. with html constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
